## Run the project

In the project directory, you can run the project by typing:

```bash 
npm start
```

## Manage flow & branches

>default branch is: Develop.

Start flow:

```bash 
git flow init
```

Open new Branch:

>All branches should be open fron the develop branch.

```bash
git flow feature start name_of_branch
```
